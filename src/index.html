Introducción a JavaScript

Abre la consola del navegador para ver los mensajes.

Una variable es como una caja donde guardamos información.

  • Etiqueta → nombre de la variable
  • Valor → lo que contiene dentro
let nombre = "Pepe";
const fechaNacimiento = "01/01/2000";
var colorFavorito = "Azul";

Reasignar:

nombre = "Juan";

Error común: crear dos cajas con el mismo nombre

let nombre = "Pepe";
let nombre = "Lola";

Tipos primitivos
  • String → "Hola"
  • Number → 33
  • Boolean → true / false
let nombre = "Pepe";
let edad = 33;
let esMayor = true;
Arrays

Una lista ordenada:

let nombres = ["Pepe", "Lola", "Matía"];
nombres[0]; // ¿Qué nos dará?
Objetos

Una ficha con información:

let persona = {
  nombre: "Pepe",
  edad: 33
};

Aritméticos
5 + 3
5 - 3
5 * 3
5 / 3
5 % 3
Comparación
5 > 3       // true
5 === "5"   // false
5 == "5"    // true (no recomendado)
Lógicos
true && false
true || false
!true

Una función es una máquina que hace algo.

Función básica
function saludar() {
  console.log("Hola");
}

saludar();
Con parámetros
function saludar(nombre) {
  console.log("Hola " + nombre);
  console.log(`Hola ${nombre}`)
}
saludar("Ana");

Sirven para tomar decisiones.

if (edad >= 18) {
  console.log("Mayor");
} else {
  console.log("Menor");
}

Repetir acciones automáticamente.

              
                for (let i = 0; i < 5; i++) {
              console.log(i);
}
